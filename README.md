
# Doctor App

This is a full-stack Doctor Appointment Booking Application.

## Features
- User authentication
- Appointment scheduling
- Doctor profiles
- Prescription management

## Tech Stack
- **Frontend:** React.js
- **Backend:** Node.js, Express.js
- **Database:** MongoDB

## Installation
1. Clone the repository:
git clone https://github.com/sivasaidwarapureddy/doctorapp.git

2. Install dependencies:
    npm install
